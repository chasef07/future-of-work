# Agent OS Knowledge

This folder is for approved durable knowledge that should be versioned in Git.

Use it for:

- Facts about people, companies, systems, and projects.
- Human preferences.
- Policies and autonomy rules.
- Repeatable playbooks.
- Improvement opportunities.

Do not put live task state, secrets, raw inbox dumps, or private operational logs
in this folder. Live state belongs in the ledger.
